# SenseCAP Watcher Add-ons for Home Assistant

This repository contains add-ons for integrating the SenseCAP Watcher device with Home Assistant.

## Add-ons

### SenseCAP Watcher AI
Visual AI assistant for SenseCAP Watcher device with camera, voice control, and display management.

[View Add-on Details](./sensecap_watcher/README.md)

## Installation

1. Go to your Home Assistant instance.
2. Navigate to **Settings** > **Add-ons** > **Add-on Store**.
3. Click the three dots in the top right corner and select **Repositories**.
4. Add the following URL: `https://github.com/judebelli1989-droid/sensecap-watcher-addon`
5. The "SenseCAP Watcher Add-ons" repository will now appear in your add-on store.
